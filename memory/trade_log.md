# Trade Log

All trades are logged here chronologically. Every buy and sell must be recorded with a reason.

## Format
```
### [DATE] [BUY/SELL] [SYMBOL]
- Action: BUY or SELL
- Shares: X
- Price: $X.XX
- Total Value: $X,XXX
- Reason: [thesis or exit reason]
- Outcome (for sells): +X% / -X%
```
